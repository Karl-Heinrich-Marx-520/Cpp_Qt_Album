# Cpp_Qt_Album | Qt电子相册入门实战项目
这是一个**专为C++/Qt初学者设计的入门级实战项目**，从零开始搭建了一个完整的桌面电子相册程序，全程基于Qt基础核心知识点，无进阶复杂内容，完美适配Qt入门学习与练手。

---

## 项目介绍
本项目是一个轻量级桌面电子相册，采用Qt Widgets开发，严格遵循**MVC分层架构**，代码结构清晰、注释完整，配套了从零到一的分步开发教程、问题解决记录、配套面试题等学习资源，是C++/Qt入门的绝佳实战项目。

### 技术栈
- 核心框架：Qt 5.15.x / Qt 6.x（兼容主流Qt版本）
- 开发语言：C++11 及以上
- 核心技术：Qt Widgets、信号槽机制、QSS样式表、双缓冲绘图、Qt属性动画、JSON持久化、文件IO操作

### 核心功能
| 功能模块 | 详细说明 |
|----------|----------|
| 项目管理 | 向导式创建/打开相册项目，JSON格式持久化存储项目配置 |
| 相册目录管理 | 左侧树形目录展示相册，右键支持新建/删除相册，自动同步磁盘文件夹 |
| 图片批量导入 | 支持多选本地图片导入，自动处理重名文件，自动复制到项目目录 |
| 缩略图展示 | 右侧图标模式展示图片缩略图，自适应窗口布局，保持宽高比 |
| 大图预览 | 双击缩略图打开大图窗口，支持鼠标滚轮缩放、拖拽平移、键盘切换上下张 |
| 幻灯片播放 | 支持淡入淡出动画切换、自动播放、进度条控制、全键盘快捷键 |
| 深色主题 | 统一的深色QSS样式，界面美观护眼，适配所有控件 |

## 编译与运行步骤（超详细，适合初学者）
1.  **克隆仓库**
    ```bash
    git clone https://github.com/Karl-Heinrich-Marx-520/Cpp_Qt_Album.git
    ```

2.  **打开项目**
    打开Qt Creator → 点击「文件」→「打开文件或项目」→ 选择仓库内 `Album/Album.pro` 文件

3.  **配置项目**
    在弹出的配置页面，选择你已安装的Qt版本和对应的编译器，点击「配置项目」

4.  **编译运行**
    点击Qt Creator左下角的「运行」按钮（绿色三角），等待编译完成后，程序会自动启动。

> 注意：如果图标资源加载失败，注释掉代码中`setIcon`相关行即可，不影响核心功能运行。

## 项目覆盖的Qt核心知识点
学完这个项目，你将掌握Qt桌面开发90%的基础常用技能：
- Qt项目创建与.pro文件配置
- 信号槽机制的规范使用（Qt5函数指针式写法）
- 常用控件：QMainWindow、QTreeWidget、QListWidget、QLabel、QPushButton、QWizard
- 布局管理：QSplitter、QVBoxLayout、QHBoxLayout、布局嵌套
- 文件IO操作：QDir、QFile、QFileInfo、JSON序列化与反序列化
- 自定义控件开发：重写绘图事件、鼠标事件、键盘事件、滚轮事件
- 双缓冲绘图：解决绘图闪烁，实现流畅的图片缩放平移
- Qt动画系统：QPropertyAnimation属性动画，实现淡入淡出特效
- Qt内存管理：QObject父子对象机制、单例模式、防止内存泄漏
- QSS样式表：自定义控件外观，实现统一的界面主题
