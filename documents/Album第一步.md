# 电子相册
## 第一步目标
1. 新建 Qt Widgets 项目
2. 主窗口**左右布局**：
   - 左边放 `QTreeWidget`（后续相册目录树）
   - 右边放 `QListWidget`（后续图片缩略图）
3. 窗口顶部做出**菜单栏**：文件、设置
4. 菜单里有：创建项目、打开项目 选项
5. 基础骨架搭好，能运行、布局正常、菜单能点

---

# 一、新建项目
1. Qt Creator → 新建项目 → **Qt Widgets 应用**
2. 项目名：`PhotoAlbum`
3. 基类选 **QMainWindow**
4. 一路下一步，完成创建

---

# 二、UI设计师页布局（核心）
双击打开 `mainwindow.ui`

### 1. 放分割器（左右布局）
1. 左边控件栏找到 **QSplitter 分割器**
2. 拖到主窗口，自动铺满整个窗口
3. 分割器改成**水平布局**（左右分栏）

### 2. 左栏放 QTreeWidget
- 从控件栏拖一个 **QTreeWidget** 到分割器左半边
- 改对象名：`treeWidgetAlbum`（方便后面代码用）

### 3. 右栏放 QListWidget
- 拖一个 **QListWidget** 到分割器右半边
- 改对象名：`listWidgetPhoto`

> 现在效果：窗口左边树控件、右边列表控件，可拖动中间分割线调整大小。

---

# 三、添加顶部菜单栏
1. 在 UI 设计器最上方 **点击「在这里输入」**
2. 输入第一个菜单：`文件(&F)`
3. 第二个菜单：`设置(&S)`

### 在「文件」菜单下加两个选项
点 文件(F) → 新建动作：
1. 第一个：`创建项目(N)`
2. 第二个：`打开项目(O)`

---

# 四、给菜单动作绑定槽（最简单方式）
1. 右键「创建项目」→ 转到槽 → 选 `triggered()` → 确定
2. Qt 自动跳到 `mainwindow.cpp`，生成空函数：
```cpp
void MainWindow::on_action创建项目_triggered()
{
    // 后续写弹出向导代码，现在先空着
}
```
3. 同理：右键「打开项目」→ 转到槽，生成另一个空槽函数。

---

# 五、直接运行测试
点运行，你现在已经有：
- 主窗口
- 顶部菜单栏：文件、设置
- 文件下有：创建项目、打开项目
- 左边树形控件、右边列表控件
- 可拖动左右分割线

✅ **这就代表第一步彻底完成**
后面第二步直接在这个骨架上写「项目向导、加载项目」就行。

---

# 总结你第一步就干了三件事
1. 建好项目 + QMainWindow 基础窗口
2. 布局：左树(QTreeWidget)、右列表(QListWidget)
3. 做好顶部菜单栏 + 两个菜单选项 + 预留好点击槽函数
